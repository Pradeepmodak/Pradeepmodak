## 🚀 Featured Engineering Projects

### 🍔 [BhookBuster – AI-Powered Food Delivery Platform](https://github.com/Pradeepmodak/BhookBuster)
> **Microservices Architecture | RabbitMQ | Redis | Docker | Socket.IO | Vector Search | Groq & Gemini AI**

* **7 Asynchronous Microservices:** Developed a resilient backend split into 7 independently deployable services communicating over **RabbitMQ** dead-letter exchanges, secured with Google OAuth and RBAC.
* **AI Search & Semantic Recommendations:** Engineered natural language food search using high-dimensional vector embeddings and RAG pipelines to deliver hyper-personalized dish suggestions.
* **Live Rider Tracking:** Implemented real-time geospatial tracking using **Socket.IO** and Leaflet Maps with offline reconnection event buffering.
* **High-Performance Infrastructure:** Cached hot catalog data with **Redis**, containerized with **Docker**, and deployed with zero-downtime CI/CD pipelines via GitHub Actions.

---

### ⚡ [ALGOPILOT – AI Competitive Coding & Mock Interview Platform](https://github.com/Pradeepmodak)
> **Online Judge Engine | Monaco Editor | Judge0 API | Groq (Llama 3.3) | Web Speech API | MongoDB**

* **Live Online Judge Engine:** Integrated Monaco Code Editor with sandboxed **Judge0 API** to execute multi-language (`C++`, `Java`, `JS`) submissions against hidden test cases.
* **Contextual AI Coding Assistant:** Embedded Groq Llama 3.3 to analyze abstract syntax, provide instant debugging hints, and generate complexity breakdowns inside the IDE.
* **Hands-Free AI Mock Interviews:** Created an automated speech-driven behavioral and technical interview simulator utilizing the **Web Speech API** for voice-to-text and text-to-speech.
* **Adaptive Roadmap Generator:** Built an analytics engine that tracks user DSA performance metrics and constructs tailored 7-, 14-, and 30-day interview preparation roadmaps.
